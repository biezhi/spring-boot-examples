# spring-boot-examples

该项目是我在学习和使用SpringBoot过程中写的一些示例程序，根据企业级需求定制。以下是涉及到的技术栈，顺序不固定。

1. Hello SpringBoot
2. 日志配置
3. 模板引擎/JSP配置
4. 拦截器配置
5. 全局异常处理
6. 国际化处理
7. 数据源配置
8. 事务管理
9. 服务注册和发现
10. 配置中心管理
11. ORM(Mybatis/JPA)
12. NOSQL使用(Redis/Mongo)
13. 分布式Session
14. 服务注册发现(Eureka/Consul)
14. 服务负载均衡(Ribbon/Feign/Zuul)
15. 服务熔断
17. 任务调度
18. 消息队列
19. 权限控制(Apache Shiro/Spring Security)
20. OAuth认证
21. Swagger API文档集成
22. 集成缓存
